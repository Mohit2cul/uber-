# Uber Clone – Frontend

React + Vite + Tailwind CSS frontend for the Uber clone project.

## Requirements
- Node.js 18+ (recommended)
- npm (comes with Node.js)

## Setup
1. Install dependencies:
	- npm install
2. Start the dev server:
	- npm run dev
3. Open the app in your browser at the URL shown in the terminal (usually http://localhost:5173).

## Build
- npm run build

## Preview Production Build
- npm run preview

## Project Structure
- src/
  - pages/: route-level pages
  - assets/: static assets
  - App.jsx: app shell
  - main.jsx: entry point

## Notes
- Tailwind is configured via tailwind.config.js and postcss.config.js.
- Update routes/pages as needed in the React app.
